# UPPER CONTROL

Aplicación Laravel + Inertia + Vue 3.

## Responsive

- Layout con sidebar colapsable en móvil (hamburger) y overlay.
- Tablas con scroll horizontal y ocultación de columnas de baja prioridad en pantallas pequeñas.
- Formularios y filtros adaptables con `flex-wrap` y controles a ancho completo en móvil.

## Stack

- Laravel 10
- Inertia.js + Vue 3
- Vite + TailwindCSS

## Desarrollo

Para construir los assets de producción:

```powershell
npm run build
```

Para entorno local:

```powershell
npm run dev
```
